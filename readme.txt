# Flask Admin Auth Example

Това е пример за Flask приложение с администраторски панел и удостоверяване на потребители (Flask-Admin + Flask-Login).

## Структура на проекта

- `app/` - Основното приложение.
- `models.py` - Модели на базата данни.
- `routes.py` - Маршрути за логин и логаут.
- `admin.py` - Конфигурация на Flask-Admin.
- `extensions.py` - Разширения като Flask-SQLAlchemy и Flask-Login.
- `templates/` - HTML шаблони.
